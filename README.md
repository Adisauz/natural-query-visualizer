# Natural Language Database Query & Visualization System

Transform natural language questions into SQL queries and automatically generate beautiful charts from your database results.

## 🚀 What it does

Ask questions in plain English like *"Show me the top 5 most popular albums with their number of songs"* and get:
- ✅ Automatically generated SQL queries
- ✅ Query execution on your database  
- ✅ Beautiful charts and visualizations
- ✅ Structured JSON responses ready for frontend integration

## 🛠️ Tech Stack

- **LangChain** - Orchestrates the AI pipeline
- **OpenAI GPT-4** - Converts natural language to SQL
- **MySQL** - Database backend (Chinook music database)
- **Matplotlib** - Automatic chart generation
- **Python** - Core implementation

## 🎯 Key Features

- Smart SQL query generation with proper table relationships
- Automatic chart type selection (bar, line, pie, scatter)
- Clean, production-ready SQL output
- Error handling and query validation
- JSON-formatted responses for easy integration
- Support for complex aggregations and joins
